[Research and Inspirations]

A lot of my inspirations in designing this project came from my current courses and likes.
A course that I'm taking this semester (as of April 2023) is CSC209, Systems Programming. This course goes in depth about using Linux systems including Bash, C programming and computer memory. In addition to being a current computer science student, I designed the home page to demonstrate a command terminal like display.

I also initially visually designed this project to mimic the theme of my favourite novel, Omniscient Reader's Viewpoint, which you can find here. It contains a lot of themes regarding constellations and book tropes, and so a lot of designs on this site feature this, from the background to the website logo.
